# bkl-project
Project for 'Kompetencje w zmieniającym się społeczeństwie'

## Setup
```console
sudo apt install python3-pip
sudo pip3 install virtualenv
virtualenv venv
source venv/bin/activate
pip install -r requirements.txt
```
